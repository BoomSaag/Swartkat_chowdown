# Swartkat Chowdown

![](_markdown/Swarkat_face.png)

Basic platformer game where you collect birds and mice for points

The game is still in very early development.

Controls are:

<i><b>Movement</b></i> = Arrow Keys

<i><b>Jump</b></i>      = Space

<i><b>Quit</b></i>        = Esc
